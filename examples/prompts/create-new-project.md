# Create A New Flato Project

Use Flato Design MCP to create a new editable design project for this brief:

```text
Create a polished one-page product launch visual for a new AI presentation tool.
Use a clear headline, supporting copy, one visual area, and a strong call to
action. Keep the design editable in Flato.
```

Follow the required bootstrap:

1. `flato_whoami`
2. `flato_create_project`
3. wait for `canWrite=true`
4. `flato_get_design_context`
5. create pages
6. re-read context and QA
