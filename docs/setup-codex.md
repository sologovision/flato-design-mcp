# Codex Setup

Install `plugins/flato-design-mcp-codex` as the Flato Design MCP Codex plugin.

The plugin bundles the hosted MCP configuration and the
`flato-design-mcp-guide` skill.

After installation, restart Codex or open a new thread, then verify:

```text
Use Flato Design MCP and call flato_whoami first.
```

If the thread cannot see `flato_*` tools after installation, start a new Codex
conversation or restart Codex. Do not inspect local OAuth or token stores as a
workaround.
